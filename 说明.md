# 开发一个个人软件开发团队的官网，使用 Next.js 14 + TypeScript，参考 [lyzeroone.com](https://www.lyzeroone.com/) 的现代极简风格，但突出以下特点：
1. **团队定位**：个人团队（非公司），强调性价比高、价格透明、售后保障。
2. **核心页面**：
   - 首页：动态英雄区（Hero Section）展示团队口号（如“用技术为您省钱”）、服务流程图、客户评价轮播。
   - 服务页：分栏展示服务类型（Web/App/小程序开发），每项标注“明码标价”（如“企业官网：¥6,666 起”）。
   - 案例页：网格布局展示过往项目，悬停效果显示技术栈和客户反馈。
   - 关于页：团队成员头像+简短介绍（突出技术背景），附售后保障政策（如“30天免费维护”）。
   - 联系页：表单+即时聊天入口（集成第三方客服工具）。
3. **设计风格**：
   - 参考 lyzeroone.com 的留白、渐变按钮、卡片阴影，但色调更活泼（如主色用蓝色系）。
   - 响应式布局，适配移动端。
   - 简约 大气 有动画
4. **技术要求**：
   - 前后端同构（Next.js API Routes 处理表单提交）。
   - 使用 TailwindCSS 或 Framer Motion 实现动画效果 。
   - 部署到 Vercel，配置自定义域名和 HTTPS。
   - 我希望我加添加案例的时候方便添加修改
5. **差异化亮点**：
   - 价格对比模块：与“传统软件公司”对比表格（我们 vs 他们：价格、响应速度、售后）。
   - 实时计价器：用户输入需求后动态计算报价（可选功能）。

# 示例组件结构（Next.js 14 App Router）
- `/app`
  - `/layout.tsx`：全局布局（导航栏+页脚）
  - `/page.tsx`：首页
  - `/services/page.tsx`：服务页
  - `/cases/[id]/page.tsx`：案例详情页（动态路由）
  - `/api/contact`：表单提交处理
- `/components`
  - `PricingCard.tsx`：价格卡片组件
  - `ServiceFlow.tsx`：服务流程步骤图
  - `TestimonialCarousel.tsx`：客户评价轮播
